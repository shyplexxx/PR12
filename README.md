# Flappy Berd 🐤

![Flappy Berd](https://images.stopgame.ru/news/2024/09/19/DFtlLph.jpg)"Flappy Berd"

## Описание проекта 📕
Данный репозторий представляет собой реализацию популярной игры **Flappy Berd** на **Python** c использованием библитеки *PyGame* . Цель этого проекта - создать простую увлекательную игру, доступную для всех уровней разработчиков, от новичков до опытных программистов

## Особенности проекта 💡

* Простота и понятие кода
* Минималистичный дизайн
* Возможность настроки игры под собственные предпочтения
* Полностью открытый исходный код

##  Системные требеования ❗
|Требования |
|-----------|
|**Python** версии 3.x | 
|Библитека **PyGame**| 

Для установки *Python* можете вопользоваться [ссылкой](https://www.python.org/downloads/)

Для установки *PyGame* зайдите в терминал и выполните комнаду: 
```pip install PyGame```


## Установка и запуск 🎮
 1. **Клонируйте репозиторий**
 ```
 git clone https://github.com/Ryoku123/FlappyBerd
 ```
 2. **Установите необходимые зависимости**
 ```
 pip install -r requirements.txt
 ```

 3. **Запустите игру**
 ```
python main.py
 ```

> [!NOTE]
> При установке, запуске и другим действия учитывайте версии всех составляющих проекта
